# Opsiary

A Python project.

## Setup

1. Clone the repository
2. Set up the virtual environment:
   ```bash
   # Activate the virtual environment
   # On macOS/Linux:
   source venv/bin/activate
   # On Windows:
   venv\Scripts\activate
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
